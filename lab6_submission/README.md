# Lab 6 Submission – CineSense Prompt Evaluation

## Submission Contents

This folder contains the complete evaluation of three prompt versions for movie review sentiment analysis.

### Files Included

#### Prompts
- `prompt_v1.txt` – Baseline prompt (simple, direct)
- `prompt_v2.txt` – Improved prompt (stricter rules, aspect analysis)
- `prompt_v3_cot.txt` – Chain-of-Thought prompt (step-by-step reasoning)

#### Testset
- `testset.csv` – 25 IMDB movie reviews with diverse difficulty levels
  - Easy reviews (clear sentiment)
  - Mixed reviews (both praise and criticism)
  - Ambiguous reviews (unclear overall sentiment)
  - Keyword trap reviews (surface words mislead)

#### Results
- `result_v1.csv` – Prompt v1 outputs on all 25 reviews
- `result_v2.csv` – Prompt v2 outputs on all 25 reviews
- `result_v3_cot.csv` – Prompt v3 CoT outputs on all 25 reviews

#### Evaluation
- `eval_v1.csv` – v1 evaluation metrics and error analysis
- `eval_v2.csv` – v2 evaluation metrics and error analysis
- `eval_v3_cot.csv` – v3 evaluation metrics and error analysis

#### Analysis
- `error_analysis.md` – Comprehensive error analysis report with:
  - Task definition
  - Testset summary
  - Detailed prompt explanations
  - Quantitative comparison (accuracy, valid JSON rate)
  - Error bucket analysis
  - Three detailed error examples
  - Reflection on CoT effectiveness
  - Final conclusions and recommendations

## Key Results

| Metric | Prompt v1 | Prompt v2 | Prompt v3 CoT |
|--------|-----------|-----------|---------------|
| **Accuracy** | 72% | 100% | 100% |
| **Valid JSON** | 100% | 100% | 100% |
| **Error Pattern** | Struggles with mixed/keyword-trap reviews | Perfect classification | Perfect classification |

### Summary

- **Prompt v1** baseline achieves 72% accuracy, failing on 7 mixed/keyword-trap reviews
- **Prompt v2** improvements (strict rules, aspect analysis) achieve 100% accuracy
- **Prompt v3** Chain-of-Thought matches v2's accuracy while improving interpretability
- **Main finding**: Structured prompting with clear rules and aspect-level analysis is crucial
